# STITT 요금제 랜딩

스팃(STITT) — 소상공인 재고·매출 관리 앱의 요금제 소개 페이지.

- **주소**: https://stitt.kr
- **호스팅**: GitHub Pages (커스텀 도메인 `stitt.kr`)
- 단일 `index.html` — CSS·JS·이미지(base64) 전부 인라인, 외부 의존성 0

## 구성

- 개인 / 팀 플랜 비교, 월간·연간 토글 (가격 카운팅 애니메이션)
- 데이/나이트 테마 (시간대 자동 전환 + 수동 토글)
- 기능 비교표 스크롤 리빌
